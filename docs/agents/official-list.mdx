## Agents

Agents made discoverable and composable through BeeAI.

| Name                                                                                                                                  | Framework | Description                                                                                                                                                                                                                                                                                 |
| ------------------------------------------------------------------------------------------------------------------------------------- | --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [agent_docs_creator](https://github.com/i-am-bee/beeai-platform/tree/main/agents/official/beeai-framework/agent-docs-creator)         | BeeAI     | The agent analyzes AI source code to generate structured, clear, and complete documentation in a consistent template, supporting multiple languages.                                                                                                                                        |
| [chat](https://github.com/i-am-bee/beeai-platform/tree/main/agents/official/beeai-framework/chat)                                     | BeeAI     | The agent is an AI-powered conversational system with memory, supporting real-time search, Wikipedia lookups, and weather updates through integrated tools.                                                                                                                                 |
| [podcast_creator](https://github.com/i-am-bee/beeai-platform/tree/main/agents/official/beeai-framework/podcast-creator)               | BeeAI     | The agent creates structured podcast-style dialogues optimized for AI-driven text-to-speech (TTS). It formats natural conversations with a lead speaker and an inquisitive co-host, ensuring realistic interruptions and follow-ups. The output is structured for seamless TTS integration. |
| [sequential_workflow](https://github.com/i-am-bee/beeai-platform/tree/main/agents/official/sequential-workflow)                       | -         | The agent orchestrates a sequence of text-processing AI agents, managing the flow of information and instructions between them.                                                                                                                                                             |

## CLI Usage

```sh
beeai run <agent_name>
```
